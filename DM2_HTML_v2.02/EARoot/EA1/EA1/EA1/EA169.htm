Appears In: Temporal Parts and Boundaries
In order to represent change over time, the IDEAS model defines temporal whole parts of individuals. Any number of temporal parts can be recognised for any given element to cover different periods of time. Those periods of time may overlap. IDEAS also defines temporal boundaries (e.g., start time, end time) as parts of whole temporal extents.
